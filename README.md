# MegaFilm Películas

Site MVP da MegaFilm Películas, empresa de venda, distribuição e instalação de películas solares, decorativas e de segurança em Curitiba/PR.

## Estrutura

- `.codex/skills/` - skills locais usadas no projeto.
- `.agents/product-marketing-context.md` - contexto de marketing para SEO, copy e posicionamento.
- `docs/site-brief.md` - briefing do site antes da implementacao.
- `public/` - logo e imagens publicas.
- `src/app/` - rotas do site em Next.js App Router.
- `src/components/` - componentes reutilizaveis.
- `src/lib/site.ts` - dados centrais de contato, navegacao, produtos e CTAs.

## Skills Locais

- `frontend-design` - direcao visual e interface.
- `seo-audit` - SEO técnico, on-page e estrutura de busca.

## Rodar localmente

```bash
npm install
npm run dev
```

No Windows PowerShell deste ambiente, use `npm.cmd` se `npm` estiver bloqueado pela execution policy.
